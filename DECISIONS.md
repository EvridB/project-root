# Ключевые архитектурные решения

1. **FastAPI + React** – современный, быстрый, типобезопасный стек.
2. **Слоистая архитектура** (роуты → сервисы → модели) для тестируемости и разделения ответственности.
3. **Атомарный UPDATE вместо блокировок** – используем `UPDATE ... WHERE` с проверкой статуса, что исключает гонки без транзакционных блокировок.
4. **Аутентификация по cookie (user_id)** – упрощает демо, легко заменить на JWT при необходимости.
5. **Docker-контейнеризация** – изолированное окружение, простой запуск одной командой.
6. **Alembic для миграций** – версионирование схемы БД.
7. **Автотесты на конкурентный доступ** – гарантируют корректную обработку гонок.
